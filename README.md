# coc-live-server

Launch a development local Server with live reload feature for static & dynamic pages.

## Install

`:CocInstall coc-live-server`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-live-server-keymap)`

## Lists

`:CocList demo_list`
